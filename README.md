🎧 Music Genre Classification using CNN
📌 Overview

This project builds a Music Genre Classifier using Convolutional Neural Networks (CNNs).
It takes raw audio files, converts them into Mel Spectrograms, and predicts the genre with solid accuracy (~78%).

Think of it like this:
👉 Your model listens to music like a human… but without bad taste.

🧠 How It Works
Pipeline:
Load audio dataset (GTZAN)
Convert audio → Mel Spectrogram
Extract features
Train CNN model
Evaluate performance
Predict genre
